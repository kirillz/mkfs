Пример с CGI

Эта страница демонстрирует функциональность CGI. Когда вы нажимаете кнопку POST, на сервер отправляется запрос HTTP POST, вызывается функция CGI. В результате любой текст, записанный (макс. 80 символов) в поле ввода, сохраняется в памяти. После этого клиенту отправляется ответ, содержащий информацию для обновления страницы.

Точно так же, когда вы нажимаете кнопку GET, HTTP-запрос GET отправляется на сервер, а затем обрабатывается в CGI. В результате содержимое следующего текстового поля заменяется текстом, сохраненным ранее при запросе POST.
